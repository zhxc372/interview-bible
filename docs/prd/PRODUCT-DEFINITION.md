# Interview Bible 产品定义 v2

> 更新时间：2026-05-10
> 来源：张小厨定义的两条产品线

---

## 核心理念

同一个JD进来，两条完全不同的路：

```
JD 输入
→ 你要什么？
   → 我想学这个岗位：学习路线生成器（Prompt A）
   → 我要面试这个岗位：面试手册生成器（Prompt B）
```

---

## 产品线一：JD驱动学习路线（Prompt A）

**解决**：小白看到JD后不知道学什么
**受众**：小白、转岗者、刚开始准备求职的人
**输出**：学习路线 + 知识模块 + 阶段计划 + 验收标准
**不做**：不生成面试答案，不写八股

**步骤**：
1. JD岗位解读
2. JD技术词翻译
3. 能力地图
4. 学习路线设计（分阶段）
5. 知识点学习卡
6. 项目练习设计
7. 学习计划（7/14/30天）
8. 下一步建议

---

## 产品线二：JD驱动面试手册（Prompt B）

**解决**：拿到JD后如何准备面试
**受众**：有基础、有面试机会、要冲刺的人
**输出**：面试作战手册 + 面试卡 + 追问防御 + 冲刺计划
**不做**：不写成长教材，不堆砌百科

**步骤**：
1. JD面试分析（显式+隐含考点）
2. 面试考点地图（P0/P1/P2/P3）
3. 面试手册TOC
4. 面试卡（30秒+2分钟+项目结合+追问X5+兜底）
5. 基础补课卡（只补面试需要的）
6. 项目深挖防御
7. 模拟面试题库
8. 面试冲刺计划（1/3/7天）
9. 打印版输出（Markdown + PDF）
