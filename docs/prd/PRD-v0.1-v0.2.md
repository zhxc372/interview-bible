# Interview Bible MVP PRD

## 1. 定位

Interview Bible MVP 是一个面试表达训练 Skill。

它把一个知识点或一个项目经历转化成：

- 可讲清
- 有证据
- 能被追问
- 不包装造假

的面试卡片。

## 2. MVP 只做

- Router 路由器
- 面试知识卡 Prompt
- 项目表达卡 Prompt
- 压力追问 Prompt
- 知识卡模板
- 项目卡模板
- MVCC 示例
- 个人 Demo 示例
- Router 测试

## 3. MVP 不做

- 简历优化器
- 八股题库生成器
- 完整模拟面试
- Web UI
- 数据库
- 长期记忆
- 自动代码仓库扫描
- AI Classroom
- Expert Explorer
- 三工具串联

## 4. 成功标准

- Router 能正确识别知识卡请求。
- Router 能正确识别项目卡请求。
- Router 能拦截混合目标。
- Router 能拦截无证据强成果表述。
- Router 能拦截 MVP 阶段代码扫描请求。
- 知识卡输出 5 字段。
- 项目卡输出 6 字段。
- 压力追问只输出 3 个问题。
- 每次输出都包含验证动作与停止条件。

## 5. 伦理边界

允许：

- 整理真实项目经历。
- 整理个人实验、课程作业、开源贡献。
- 设计可复现 Demo。
- 生成压力追问。
- 优化表达结构。

禁止：

- 编造公司项目。
- 编造上线数据。
- 编造团队角色。
- 编造用户反馈。
- 编造 PR / Commit / 日志。
- 把个人实验伪装成商业生产。
- 把本地压测写成生产指标。

## 6. 最小验收案例

1. MVCC 知识卡跑通。
2. 个人 Demo 项目卡跑通。
3. 无证据高并发表述被拦截。
4. 混合目标被拦截。
5. 代码扫描请求在 MVP 阶段被拦截。
